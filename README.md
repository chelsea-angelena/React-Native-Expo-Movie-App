## Getting Started

- npm install
- npm run ios

### Prerequisites - MacOS

- npm
  ```sh
  npm install npm@latest -g
  ```
- node
  ```sh
  brew install node
  ```
- xcode (App store)
- expo cli
- Firebase/firestore account (free)
